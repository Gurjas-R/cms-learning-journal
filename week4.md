## Content Insights
This week I learned that web development follows a structured process rather than just building a website and uploading it. The process includes planning, development, testing, and deployment. I learned that developers usually work in different environments such as local, staging, and production to avoid breaking the live website. Version control systems like Git allow developers to track changes, work in teams, and revert to previous versions if something breaks. Another important thing was that migrating a WordPress site involves moving both the website files and the database, which showed me that dynamic websites are more complex than static websites.

## Career / Employability / Learning Insights
This week helped me understand how professional developers manage projects and work in teams. Version control is an important skill because it allows multiple developers to work on the same project without overwriting each other’s work. I also learned and tested in the practical how development workflows and migration are important because websites are often moved between servers or environments. These skills are important for working in web development because most companies use version control and structured workflows when building websites.
