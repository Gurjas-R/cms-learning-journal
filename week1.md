# Week 1 – Basic Web Design

## Learning Activities & Resources
For this weeks topic I focused on revising and relearning basic HTML and Git Commits to build the simple static website about a startup idea for practical 1. I read the bootstrap documents and used the template as suggested by the practical instruction then worked off it to add images and text creating a simple static 3 page website. I used a W3 schools document to make the nav bar to connect the 3 pages as I had forgotten how to create a nav bar. Then published it using Git pages and used the guide for Git pages to do son as I had never done so before. 

Resources I used were The Bootstrap template, GitHub Pages guide, W3Schools HTML tutorials.

https://getbootstrap.com/docs/ - Bootstrap
https://docs.github.com/en/pages - Github Pages
https://www.w3schools.com/howto/howto_js_topnav.asp - W3 Schools

## Content Insights
This week I learned web development is not just writing code, it is about the whole design process of a website. In particular on how a website meets a specific goal and is usable for the selected target audience. The first lecture emphasises how this subject is designed to simulate real industry workflows, where developers are expected to figure out solutions individualy rather than following a step by step plan. I also revised HTML and CSS, HTML the structure of the website and CSS is the design and layout of a website. This week in particular I learned about Bootstrap for the first time and how to use Github pages, this allowed me to gain insight on how a real website would be deployed for companies and not just locally hosted using xxamp or other local hosting softwares.

## Career / Employability / Learning Insights
One important thing I learned is that employers expect developers to be able to learn independently and solve problems without being given exact specific instructions or guided constantly. This module has given me an idea on how real industry enviornments would feel and work. I have come to a much better understanding of what is required of me and how I should complete it when I enter the real working industry for my ideal career choice of development. Learning how to use GitHub and GitHub Pages is also useful for my future employability because it allows me to share work, collaborate with others, and create a portfolio of projects to show on my resume.
