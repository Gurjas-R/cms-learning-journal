# Week 3 – WordPress CMS

## Learning Activities & Resources
This module I learned how to use WordPress to build a website for my startup idea. I created pages and added them to a navigation menu so users can move between pages. I read the guide on WordPress themes, plugins, and menus, and I read some documentation to understand how WordPress works as a CMS.

## Content Insights
This week I learned how WordPress separates content from design. The content such as text and pages is stored in a database, while the theme controls the layout and appearance of the site. This means I can change the design of the website without changing the content. I also learned that there are plugins like Joomla has that can add features without needing to write code, which makes development faster. After using both Joomla and WordPress, I found WordPress easier to use and easier to customise.

## Career / Employability / Learning Insights
This week helped me understand how CMS platforms are used in real jobs. A developer can build the site and set everything up, and then the client can log in and edit the content without changing the code. I also learned that developers often customise themes instead of building everything from scratch, which saves time. This week helped me understand how CMS skills are useful for web development jobs.
