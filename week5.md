## Content Insights
This week I learned that a child theme allows me to customise a WordPress website without editing the parent theme directly. WordPress loads the parent theme first and then loads the child theme, which allows the child theme to override styles and functions. This is important because if the parent theme updates, the customisations in the child theme will not be lost. Also responsive web design is important because websites must work on different screen sizes such as phones, tablets, and desktops. Responsive design uses flexible layouts and media queries so the layout can adjust depending on screen size.

## Career / Employability / Learning Insights
This week helped me understand how WordPress developers customise websites in real projects. Instead of building a theme from scratch, developers often customise an existing theme using a child theme. This saves time and allows updates to be made safely. I also learned that responsive design is an important skill because many users access websites on mobile devices. These skills are useful for web development jobs because WordPress is considered a industry standard and many clients need websites that work on different devices.
